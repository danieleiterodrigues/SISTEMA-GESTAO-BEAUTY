# JS Beauty Studio - Sistema de Gestão

Sistema completo de gestão para o JS Beauty Studio, gerenciando financeiro, comercial, atendimentos e administração.

## Stack Tecnológica

### Backend
- **Framework**: NestJS
- **Linguagem**: TypeScript
- **Banco de Dados**: PostgreSQL (via Prisma ORM)
- **Autenticação**: JWT

### Frontend
- **Framework**: React + Vite
- **Linguagem**: TypeScript
- **Estilo**: TailwindCSS
- **Estado**: Zustand + TanStack Query

### Infraestrutura
- Docker & Docker Compose

## Configuração Inicial

### Pré-requisitos
- Node.js 18+
- Docker & Docker Compose

### Instalação

1. Configure o ambiente:
   ```bash
   # Inicie o banco de dados
   docker-compose up -d postgres
   ```

2. (Pendente) Inicialização do Backend e Frontend via scripts de setup.
